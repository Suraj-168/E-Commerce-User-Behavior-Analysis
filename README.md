\# E-Commerce User Behavior Analysis



\## Overview



This project analyzes user behavior on an e-commerce platform using Excel and MySQL. Analysis of a 5-stage e-commerce conversion funnel (page view → add to cart → checkout → payment → purchase) to identify where users drop off, which traffic sources convert best, and where revenue is actually coming from. Built using both SQL and Excel to cross-validate results.



\## Tools Used



\- MySQL

\- Microsoft Excel

\- Pivot Tables

\- Charts



\## Analysis Performed



\- User funnel analysis

\- Traffic source analysis

\- Conversion rates

\- Product performance

\- Revenue analysis

\- Customer journey timing
